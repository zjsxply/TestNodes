# TestNodes

收集互联网上公开的科学上网节点，并进行测试

支持测试 `Tcping`、真连接延迟、Speedtest，支持检测入口 IP 地区、出口 IP 地区、奈飞是否解锁

目前有大约 200 个可用节点，覆盖 40 个国家或地区。
